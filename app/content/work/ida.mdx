Ida.io is a multi-lingual site for Ida.io, an AI farming start up that is bespoke hardware and AI to help farmers manage their cow herds.

Built using Next.js, Sanity, and coded in Typescript to create a modern, performant static site with a strong emphasis on UX for content creators.
